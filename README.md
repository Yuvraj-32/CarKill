# Carkill.io

🚗💥 A fast-paced 3D multiplayer vehicular combat arena built with Three.js and Socket.io.

## Features
- 3D chase-camera perspective
- 4 vehicle types (Car, Pickup, Van, Tank) with front spikes
- Realistic environment with trees, sand patches, and pit traps
- Real-time multiplayer via WebSockets
- Leaderboard, kill feed, speedometer HUD

## Run Locally
```bash
npm install
npm start
```
Open http://localhost:3000

## Tech Stack
- **Frontend**: Three.js (3D), HTML5, CSS3, Vanilla JS (ES Modules)
- **Backend**: Node.js, Express, Socket.io
- **Deployment**: Fly.io (Docker)
